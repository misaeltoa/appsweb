Frameworks

AngularJS

AngularJS es un framework de JavaScript mantenido por Google y por una comunidad de desarrolladores, que sirve para crear aplicaciones web dinámicas. Permite usar HTML como lenguaje de plantilla y extender la sintaxis de HTML para expresar los componentes de la aplicación de manera clara y concisa.

  • Desarrollado por: Google
  • Lenguaje: JavaScript
  • Última versión: 1.8.x
  • Licencia: MIT
  • Sitio web: angularjs.org

Laravel

Laravel es un framework de PHP diseñado para el desarrollo de aplicaciones web siguiendo el patrón de diseño MVC. Es conocido por su elegante sintaxis y herramientas poderosas como el sistema de migraciones, el ORM Eloquent y el motor de plantillas Blade.

  • Desarrollado por: Taylor Otwell
  • Lenguaje: PHP
  • Última versión: 9.x
  • Licencia: MIT
  • Sitio web: laravel.com

Node.js

Node.js es un entorno de ejecución de JavaScript que permite ejecutar código del lado del servidor. Está construido sobre el motor V8 de Google Chrome y es conocido por su capacidad para manejar aplicaciones de alta concurrencia con un solo hilo utilizando un modelo de E/S no bloqueante y orientado a eventos.

  • Desarrollado por: Ryan Dahl
  • Lenguaje: JavaScript
  • Última versión: 20.x
  • Licencia: MIT
  • Sitio web: nodejs.org

Spring

Spring es un framework de Java utilizado para crear aplicaciones empresariales robustas. Proporciona infraestructura de soporte para desarrollar aplicaciones en Java, facilitando el desarrollo y prueba de aplicaciones de nivel empresarial.

  • Desarrollado por: Pivotal Software
  • Lenguaje: Java
  • Última versión: 5.x
  • Licencia: Apache License 2.0
  • Sitio web: spring.io
